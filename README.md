# 📚 Tài liệu học đồ án Quản lý Nhà sách (Tkinter)

Trang web học chi tiết và tường tận cho đồ án **quanlynhasach** — phiên bản Tkinter.

🔗 **Đồ án nguồn:** https://github.com/huyhandsome6996/quanlynhasach

## Mục tiêu

Giải thích cho người mới học / mất gốc mọi thứ trong đồ án:
- Thuật toán (DSLK đôi, Merge Sort, Stack LIFO, Queue FIFO)
- OOP (kế thừa + đa hình với 5 lớp con)
- Tkinter (cú pháp từng widget: Treeview, Combobox, Entry, Button, ...)
- SQLite (CRUD, parameterized query, phân quyền)
- 9 chức năng chi tiết (CN01 → CN09)

## Cách sử dụng

Mở file `index.html` bằng trình duyệt. Học theo thứ tự trong sidebar trái.

## Lộ trình học (4 nhóm, 19 bài)

1. **Kiến thức nền** (4 bài): Python → OOP → Tkinter → SQLite
2. **Thuật toán & Cấu trúc dữ liệu** (4 bài): DSLK → Merge Sort → Stack → Queue
3. **9 Chức năng** (9 bài): CN01 → CN09
4. **Tổng thể** (2 bài): main.py + QuanLyTrungTam, Phân chia công việc
