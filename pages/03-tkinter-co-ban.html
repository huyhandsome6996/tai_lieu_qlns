🖼️ Bài 3: Tkinter cơ bản

Tkinter là thư viện tạo giao diện đồ họa (GUI) có sẵn trong Python. Không cần cài đặt — import tkinter là dùng được.

ℹ️ Tại sao dùng Tkinter?

Tkinter đơn giản, dễ học, có sẵn trong Python. Đây là phiên bản chính nhóm chọn để thi.

📑 Mục lục
  1. Tạo cửa sổ đầu tiên
  2. Label — nhãn chữ
  3. Entry — ô nhập chữ
  4. Button — nút bấm
  5. Combobox — dropdown chọn
  6. Treeview — bảng dữ liệu
  7. messagebox — hộp thoại thông báo
  8. Toplevel — cửa sổ con
  9. Sự kiện (events) + bind
  10. Layout: pack / grid / place
  11. Ví dụ hoàn chỉnh: form đăng nhập

1. Tạo cửa sổ đầu tiên

import tkinter as tk

# Tạo cửa sổ chính
root = tk.Tk()

# Đặt tiêu đề cửa sổ
root.title("Chương trình đầu tiên")

# Đặt kích thước (rộng x cao)
root.geometry("400x300")

# Chạy vòng lặp chính (lắng nghe sự kiện)
root.mainloop()
🎨 Hình minh hoạ: cửa sổ Tk() trông như thế nào

Đây là hình minh hoạ (mockup) của cửa sổ Tkinter. Trên Linux/Mac/Windows giao diện có thể hơi khác (theme hệ điều hành), nhưng cấu trúc cơ bản giống hệt:

Chương trình đầu tiên
👋 Xin chào Tkinter!
Bấm tôi
Sẵn sàng

Hình 1: Cửa sổ tk.Tk() với 1 Label + 1 Button. Đây là kết quả khi chạy code mẫu ở trên.

Chạy code trên → xuất hiện cửa sổ trắng 400×300 có tiêu đề.

⚠️ Quên mainloop() = cửa sổ không hiện

root.mainloop() là dòng lệnh "chạy mãi mãi" để chờ user bấm nút, gõ phím, click chuột. Nếu quên → chương trình kết thúc ngay → cửa sổ biến mất.

2. Label — nhãn chữ

Label dùng để hiển thị chữ (hoặc ảnh) — không cho user sửa.

import tkinter as tk

root = tk.Tk()
root.title("Ví dụ Label")
root.geometry("400x200")

# Label cơ bản
label1 = tk.Label(root, text="Xin chào!")
label1.pack(pady=20)                  # pack() = đặt widget vào cửa sổ

# Label có định dạng chữ
label2 = tk.Label(
    root,
    text="Chữ to, đỏ, đậm",
    font=("Arial", 20, "bold"),       # font chữ
    fg="red",                         # foreground = màu chữ
    bg="yellow"                       # background = màu nền
)
label2.pack(pady=10)

# Label căn lề
label3 = tk.Label(
    root,
    text="Căn giữa",
    fg="blue",
    font=("Times New Roman", 14),
    width=20,
    height=2,                         # chiều cao tính theo dòng chữ
    anchor="center"                   # căn giữa: center, w (trái), e (phải)
)
label3.pack()

root.mainloop()

Các tham số Label thường dùng

Tham số Ý nghĩa Ví dụ

fgMàu chữ (foreground)"red", "#FF0000"
bgMàu nền (background)"yellow"

heightCao (số dòng)2

3. Entry — ô nhập chữ

Entry là ô nhập liệu 1 dòng (như ô nhập tên đăng nhập).

import tkinter as tk

root = tk.Tk()
root.title("Ví dụ Entry")
root.geometry("400x200")

# Label hướng dẫn
tk.Label(root, text="Tên đăng nhập:").pack(pady=5)

# Entry cơ bản
entry_ten = tk.Entry(root, width=30)
entry_ten.pack(pady=5)

# Entry cho mật khẩu (hiện dấu *)
tk.Label(root, text="Mật khẩu:").pack(pady=5)
entry_mk = tk.Entry(root, width=30, show="*")     # show="*" → mỗi ký tự thành *
entry_mk.pack(pady=5)

# Hàm đọc giá trị Entry
def lay_gia_tri():
    ten = entry_ten.get()                # get() → lấy nội dung Entry
    mk = entry_mk.get()
    print(f"Tên: {ten}, MK: {mk}")

# Nút để gọi hàm
tk.Button(root, text="In giá trị", command=lay_gia_tri).pack(pady=10)

root.mainloop()

Các thao tác Entry

# Lấy nội dung Entry
gia_tri = entry.get()

# Xóa toàn bộ Entry
entry.delete(0, tk.END)              # 0 = vị trí bắt đầu, tk.END = vị trí cuối

# Xóa từ vị trí 2 đến 4
entry.delete(2, 4)

# Chèn chữ vào Entry
entry.insert(0, "Chữ ban đầu")       # 0 = vị trí chèn

# Chèn chữ vào cuối
entry.insert(tk.END, "Thêm cuối")

# Đặt focus (con trỏ chuột tự vào Entry)
entry.focus()

# Vô hiệu hóa Entry (không cho sửa)
entry.config(state="disabled")

# Kích hoạt lại
entry.config(state="normal")

4. Button — nút bấm

import tkinter as tk

def xu_ly_click():
    print("Bạn đã bấm nút!")
    # Đổi chữ nút sau khi bấm
    nut.config(text="Đã bấm ✓")

root = tk.Tk()
root.title("Ví dụ Button")
root.geometry("300x200")

# Button cơ bản
nut = tk.Button(
    root,
    text="Bấm tôi!",
    command=xu_ly_click,                 # command = hàm chạy khi bấm
    bg="#4CAF50",                        # màu nền xanh lá
    fg="white",                          # chữ trắng
    font=("Arial", 12, "bold"),
    padx=20,                             # đệm ngang
    pady=10                              # đệm dọc
)
nut.pack(pady=30)

root.mainloop()
⚠️ command=ten_ham KHÔNG có dấu ()

Sai: command=xu_ly_click() → gọi hàm ngay khi tạo Button!

Đúng: command=xu_ly_click → gọi hàm khi user bấm.

Đổi command động (sau khi tạo Button)

# Ban đầu
nut = tk.Button(root, text="Bấm", command=ham_A)

# Đổi sang hàm B
nut.config(command=ham_B)

# Bỏ command (nút không làm gì khi bấm)
nut.config(command=lambda: None)

5. Combobox — dropdown chọn

Combobox nằm trong module ttk (themed Tkinter). Cần from tkinter import ttk.

import tkinter as tk
from tkinter import ttk

def khi_chon(event):
    gia_tri = combo.get()
    print(f"Bạn đã chọn: {gia_tri}")

root = tk.Tk()
root.title("Ví dụ Combobox")
root.geometry("300x150")

tk.Label(root, text="Chọn loại hàng:").pack(pady=5)

# Tạo Combobox với các lựa chọn
combo = ttk.Combobox(
    root,
    values=["Sách", "Tạp chí", "Báo giấy", "Luận văn", "Bản thảo"],
    state="readonly"                       # không cho gõ tay, chỉ chọn
)
combo.set("Sách")                          # giá trị mặc định
combo.pack(pady=10)

# Bắt sự kiện khi user chọn giá trị mới
combo.bind("<<ComboboxSelected>>", khi_chon)

root.mainloop()

6. Treeview — bảng dữ liệu

Treeview là bảng có cột + hàng — giống Excel. Đây là widget quan trọng nhất trong đồ án vì bảng danh sách mặt hàng dùng nó.

import tkinter as tk
from tkinter import ttk

root = tk.Tk()
root.title("Ví dụ Treeview")
root.geometry("600x300")

# Tạo Treeview với 4 cột
bang = ttk.Treeview(root, columns=("ma", "ten", "loai", "gia"), show="headings")

# Đặt tiêu đề + độ rộng cho từng cột
bang.heading("ma", text="Mã số")
bang.heading("ten", text="Tên sản phẩm")
bang.heading("loai", text="Loại")
bang.heading("gia", text="Giá bán")

bang.column("ma", width=80, anchor="center")
bang.column("ten", width=200, anchor="w")
bang.column("loai", width=100, anchor="center")
bang.column("gia", width=100, anchor="e")

bang.pack(fill="both", expand=True, padx=10, pady=10)

# Thêm dữ liệu vào bảng
bang.insert("", tk.END, values=("MH001", "Truyện Kiều", "Sách", "55000"))
bang.insert("", tk.END, values=("MH002", "Tạp chí KH", "Tạp chí", "31500"))
bang.insert("", tk.END, values=("MH003", "Báo Tuổi Trẻ", "Báo giấy", "5000"))

# Lấy dòng được chọn
def lay_dong_chon():
    chon = bang.selection()              # trả list các ID dòng được chọn
    if chon:
        gia_tri = bang.item(chon[0])["values"]   # lấy values của dòng đầu
        print("Dòng được chọn:", gia_tri)

tk.Button(root, text="Lấy dòng chọn", command=lay_dong_chon).pack(pady=5)

root.mainloop()

7. messagebox — hộp thoại thông báo

messagebox dùng để hiện thông báo nhỏ (thành công, lỗi, cảnh báo...).

import tkinter as tk
from tkinter import messagebox

root = tk.Tk()
root.title("Ví dụ messagebox")
root.geometry("300x200")

def hien_thong_bao():
    # Thông báo thành công
    messagebox.showinfo("Thông báo", "Thao tác thành công!")

def hoi_xac_nhan():
    # Hỏi Yes/No — trả về True/False
    tra_loi = messagebox.askyesno("Xác nhận", "Bạn có chắc muốn xóa?")
    if tra_loi:
        print("Đã xóa")
    else:
        print("Hủy xóa")

def bao_loi():
    # Báo lỗi
    messagebox.showerror("Lỗi", "Không tìm thấy sản phẩm!")

def canh_bao():
    # Cảnh báo
    messagebox.showwarning("Cảnh báo", "Số lượng tồn kho thấp!")

tk.Button(root, text="Thông báo", command=hien_thong_bao).pack(pady=5)
tk.Button(root, text="Hỏi Yes/No", command=hoi_xac_nhan).pack(pady=5)
tk.Button(root, text="Báo lỗi", command=bao_loi).pack(pady=5)
tk.Button(root, text="Cảnh báo", command=canh_bao).pack(pady=5)

root.mainloop()

8. Toplevel — cửa sổ con

Toplevel dùng để tạo cửa sổ phụ (như form thêm/sửa) — riêng biệt với cửa sổ chính.

import tkinter as tk

def mo_cua_so_con():
    top = tk.Toplevel(root)               # Toplevel = cửa sổ con
    top.title("Cửa sổ con")
    top.geometry("300x200")
    top.grab_set()                        # grab_set = modal (không cho click cửa sổ chính)

    tk.Label(top, text="Đây là cửa sổ con", font=("Arial", 14)).pack(pady=50)

    tk.Button(top, text="Đóng", command=top.destroy).pack()

root = tk.Tk()
root.title("Cửa sổ chính")
root.geometry("400x300")

tk.Button(root, text="Mở cửa sổ con", command=mo_cua_so_con).pack(pady=100)

root.mainloop()

Các thao tác Toplevel

# Tạo
top = tk.Toplevel(root)

# Đặt kích thước + vị trí
top.geometry("400x300+100+100")          # rộng×cao+x+y (x, y từ góc trái trên màn hình)

# Modal — không cho click cửa sổ chính khi đang mở cửa sổ con
top.grab_set()

# Chờ đến khi cửa sổ con đóng
root.wait_window(top)                     # block cho đến khi top.destroy() được gọi

# Đóng cửa sổ con
top.destroy()

# Ẩn (không hủy) — có thể hiện lại
top.withdraw()

# Hiện lại
top.deiconify()

9. Sự kiện (events) + bind

bind dùng để gắn sự kiện (bấm phím, click chuột...) vào widget.

import tkinter as tk

def xu_ly_phim_enter(event):
    print("Bạn đã bấm Enter!")

def xu_ly_click_chuot(event):
    print(f"Bạn click chuột tại tọa độ: {event.x}, {event.y}")

def xu_ly_phim_bat_ky(event):
    print(f"Bạn bấm phím: {event.char} (mã: {event.keysym})")

root = tk.Tk()
root.geometry("400x300")

entry = tk.Entry(root, width=30)
entry.pack(pady=20)

# Gắn phím Enter khi focus trong Entry
entry.bind("<Return>", xu_ly_phim_enter)              # Return = Enter

# Gắn phím bất kỳ
entry.bind("<Key>", xu_ly_phim_bat_ky)

# Gắn click chuột vào cửa sổ
root.bind("<Button-1>", xu_ly_click_chuot)           # Button-1 = chuột trái

root.mainloop()

Mã sự kiện thường dùng

MãÝ nghĩa
<Return>Phím Enter
<Escape>Phím Esc
<Tab>Phím Tab
<Key>Bất kỳ phím nào
<Button-1>Click chuột trái
<Button-2>Click chuột giữa
<Button-3>Click chuột phải
<Double-Button-1>Double-click chuột trái
<FocusIn>Widget nhận focus
<FocusOut>Widget mất focus

10. Layout: pack / grid / place

Có 3 cách xếp widget trong cửa sổ. Đồ án dùng chủ yếu pack và grid.

pack() — xếp tuần tự

# Mặc định xếp từ trên xuống
tk.Label(root, text="Dòng 1").pack()
tk.Label(root, text="Dòng 2").pack()
tk.Label(root, text="Dòng 3").pack()

# Xếp từ trái sang
tk.Label(root, text="A").pack(side="left")
tk.Label(root, text="B").pack(side="left")
tk.Label(root, text="C").pack(side="left")

# side: top, bottom, left, right
# fill: "x" (lấp đầy ngang), "y" (lấp đầy dọc), "both"
# expand: True → widget lấp đầy không gian còn lại

tk.Label(root, text="Lấp đầy").pack(fill="both", expand=True)

grid() — xếp dạng lưới (như bảng)

# Tạo form đăng nhập bằng grid
tk.Label(root, text="Tên:").grid(row=0, column=0, sticky="e", padx=5, pady=5)
tk.Entry(root).grid(row=0, column=1, padx=5, pady=5)

tk.Label(root, text="Mật khẩu:").grid(row=1, column=0, sticky="e", padx=5, pady=5)
tk.Entry(root, show="*").grid(row=1, column=1, padx=5, pady=5)

tk.Button(root, text="Đăng nhập").grid(row=2, column=0, columnspan=2, pady=10)
# row: dòng (bắt đầu từ 0)
# column: cột (bắt đầu từ 0)
# columnspan: gộp nhiều cột
# rowspan: gộp nhiều dòng
# sticky: "n" (trên), "s" (dưới), "e" (phải), "w" (trái), hoặc kết hợp "nsew"
# padx, pady: đệm ngang/dọc
⚠️ Không trộn pack và grid

Trong cùng cửa sổ, dùng HOẶC pack HOẶC grid — không trộn lẫn. Nếu trộn → Tkinter báo lỗi và sập chương trình.

place() — đặt tọa độ tuyệt đối

# Đặt widget tại tọa độ (x, y) tính từ góc trái trên
tk.Button(root, text="Nút").place(x=50, y=100, width=100, height=30)

# x: tọa độ ngang
# y: tọa độ dọc
# width, height: kích thước
# relx, rely: tọa độ theo % (0.0 - 1.0) so với cửa sổ

11. Ví dụ hoàn chỉnh: form đăng nhập

Đây là version đơn giản của dang_nhap_ui.py trong đồ án:

import tkinter as tk
from tkinter import messagebox

class FormDangNhap:
    """Form đăng nhập đơn giản."""

    def __init__(self, root):
        # Tạo cửa sổ con (Toplevel)
        self.top = tk.Toplevel(root)
        self.top.title("Đăng nhập")
        self.top.geometry("350x200")
        self.top.grab_set()                    # modal

        # Tiêu đề
        tk.Label(
            self.top, text="ĐĂNG NHẬP HỆ THỐNG",
            font=("Arial", 14, "bold"), fg="#1e3a5f"
        ).grid(row=0, column=0, columnspan=2, pady=15)

        # Label + Entry cho tên đăng nhập
        tk.Label(self.top, text="Tên đăng nhập:").grid(
            row=1, column=0, sticky="e", padx=10, pady=5
        )
        self.entry_ten = tk.Entry(self.top, width=25)
        self.entry_ten.grid(row=1, column=1, padx=10, pady=5)
        self.entry_ten.focus()                 # focus tự động

        # Label + Entry cho mật khẩu
        tk.Label(self.top, text="Mật khẩu:").grid(
            row=2, column=0, sticky="e", padx=10, pady=5
        )
        self.entry_mk = tk.Entry(self.top, width=25, show="*")
        self.entry_mk.grid(row=2, column=1, padx=10, pady=5)

        # Nút đăng nhập
        self.btn_dn = tk.Button(
            self.top, text="Đăng nhập",
            command=self.xu_ly_dang_nhap,
            bg="#4CAF50", fg="white",
            font=("Arial", 10, "bold"),
            width=15
        )
        self.btn_dn.grid(row=3, column=0, columnspan=2, pady=15)

        # Gắn phím Enter cho ô mật khẩu
        self.entry_mk.bind("<Return>", lambda e: self.xu_ly_dang_nhap())

        # Biến lưu người dùng sau khi đăng nhập
        self.nguoi_dung = None

    def xu_ly_dang_nhap(self):
        ten = self.entry_ten.get().strip()
        mk = self.entry_mk.get().strip()

        if not ten or not mk:
            messagebox.showerror("Lỗi", "Vui lòng nhập đầy đủ!")
            return

        # Giả lập kiểm tra (thực tế gọi hàm check DB)
        if ten == "admin" and mk == "123":
            self.nguoi_dung = {"ten": ten, "vai_tro": "admin"}
            self.top.destroy()                 # đóng form
        else:
            messagebox.showerror("Lỗi", "Sai tên hoặc mật khẩu!")
            self.entry_mk.delete(0, tk.END)    # xóa mật khẩu

    def lay_ten(self):
        return self.entry_ten.get().strip()

    def lay_mat_khau(self):
        return self.entry_mk.get().strip()


# Chạy thử
if __name__ == "__main__":
    root = tk.Tk()
    root.title("Ứng dụng chính")
    root.geometry("400x300")
    root.withdraw()                            # ẩn root, chỉ hiện form đăng nhập

    form = FormDangNhap(root)
    root.wait_window(form.top)                 # chờ form đóng

    if form.nguoi_dung:
        print("Đăng nhập thành công:", form.nguoi_dung)
        root.deiconify()                       # hiện root
        root.mainloop()
    else:
        root.destroy()                         # đóng luôn
💡 Phân tích code trên
  • Toplevel(root) — tạo cửa sổ con từ root
  • grab_set() — modal, không cho bấm root
  • grid(row=, column=) — xếp dạng lưới
  • columnspan=2 — nút chiếm 2 cột
  • show="*" — mật khẩu hiện dấu *
  • .focus() — con trỏ tự vào ô
  • .bind("<Return>", ...) — Enter = bấm nút
  • .get() — lấy giá trị Entry
  • .delete(0, tk.END) — xóa Entry
  • .destroy() — đóng cửa sổ
  • root.wait_window(form.top) — chờ form đóng
  • root.withdraw() / deiconify() — ẩn / hiện root
