📋 CN02: Đọc danh sách sản phẩm

Đây là chức năng phổ biến nhất — đọc toàn bộ sản phẩm từ DSLK đôi và hiển thị lên bảng Treeview. Cứ mỗi lần thêm/sửa/xóa/undo/redo, hệ thống đều gọi lại chức năng này để cập nhật bảng.

Nhóm 2 — Đọc danh sách: Khi đăng nhập thành công, bảng dữ liệu hiển thị toàn bộ sản phẩm (mã, tên, loại, giá, tồn kho, giá bán, thông tin riêng). Định dạng giá tiền có dấu phẩy hàng nghìn, dễ đọc.

2. File _tt: lay_danh_sach

File thuật toán rất ngắn — chỉ gọi hàm chuyen_thanh_danh_sach() của DSLK đôi (đã học ở bài DSLK).

# Import lớp DSLK đôi từ thư viện thuật toán.
from loi_thuat_toan.danh_sach_lien_ket import DoublyLinkedList


def lay_danh_sach(danh_sach: DoublyLinkedList):
    """
    Đọc toàn bộ mặt hàng trong DSLK đôi và trả về list dict.

    Tham số:
        danh_sach : đối tượng DoublyLinkedList chứa các MatHang.

    Trả về:
        List[dict] — mỗi dict là 1 mặt hàng.
        Trả về [] nếu lỗi hoặc DSLK rỗng.
    """
    try:
        # Nếu DSLK chưa khởi tạo → trả về danh sách rỗng.
        if danh_sach is None:
            return []
        # Gọi hàm có sẵn của DoublyLinkedList để chuyển DSLK → list dict.
        return danh_sach.chuyen_thanh_danh_sach()
    except Exception as loi:
        # Lỗi bất kỳ → in ra + trả về danh sách rỗng để UI không bị crash.
        print(f'[LOI doc_danh_sach] {loi}')
        return []

Giải thích cú pháp

2.1. Type hint danh_sach: DoublyLinkedList

Dấu hai chấm : khai báo kiểu dữ liệu tham số. Đây chỉ là gợi ý (Python không ép buộc) — giúp người đọc biết tham số phải là DoublyLinkedList, không phải int hay str.

2.2. Try-except bảo vệ UI

Khi gọi hàm đọc dữ liệu, có thể xảy ra lỗi (DSLK chưa khởi tạo, Node bị tham chiếu rác, v.v.). Bọc trong try-except để khi lỗi, hàm trả về [] thay vì crash — UI sẽ hiển thị bảng trống thay vì sập toàn app.

⚠️ Nguyên tắc: Không bao giờ để UI crash

Mọi hàm thuật toán có thể gây lỗi đều phải bọc try-except. Khi lỗi:

  1. In lỗi ra console (print(...)) để dev debug.
  2. Trả về giá trị an toàn ([], {}, None, False).

UI chỉ cần kiểm tra giá trị trả về — không cần try-except lại.

3. File _ui: BangSanPhamUI (Treeview)

Đây là phần quan trọng nhất — tạo bảng Treeview để hiển thị danh sách sản phẩm. Treeview là widget tiêu chuẩn của Tkinter cho bảng dữ liệu.

import tkinter as tk
from tkinter import ttk


class BangSanPhamUI:
    """Bảng dữ liệu sản phẩm dùng Treeview."""

    # Định nghĩa cột của bảng — dùng tuple để gọn.
    COT = ('ma', 'ten', 'loai', 'gia_cb', 'ton', 'gia_ban', 'rieng')

    def __init__(self, parent):
        """Tạo bảng Treeview trong parent."""
        # Tạo Treeview với 7 cột, show='headings' để ẩn cột #0 (cây).
        self.tree = ttk.Treeview(parent, columns=self.COT, show='headings', height=15)

        # Định nghĩa header (tiêu đề cột) + căn lề + độ rộng.
        self.tree.heading('ma', text='Mã số')
        self.tree.heading('ten', text='Tên sản phẩm')
        self.tree.heading('loai', text='Loại')
        self.tree.heading('gia_cb', text='Giá cơ bản')
        self.tree.heading('ton', text='Tồn kho')
        self.tree.heading('gia_ban', text='Giá bán (đã tính)')
        self.tree.heading('rieng', text='Thông tin riêng')
        self.tree.column('ma', width=70, anchor='center')
        self.tree.column('ten', width=200)
        self.tree.column('loai', width=80, anchor='center')
        self.tree.column('gia_cb', width=100, anchor='e')
        self.tree.column('ton', width=70, anchor='center')
        self.tree.column('gia_ban', width=110, anchor='e')
        self.tree.column('rieng', width=250)

        # Tạo scrollbar dọc để cuộn danh sách dài.
        scrollbar = ttk.Scrollbar(parent, orient='vertical', command=self.tree.yview)
        self.tree.configure(yscrollcommand=scrollbar.set)

        # Đặt widget vào parent: tree bên trái, scrollbar bên phải.
        self.tree.pack(side='left', fill='both', expand=True)
        scrollbar.pack(side='right', fill='y')

    def xoa_tat_ca_dong(self):
        """Xóa toàn bộ dòng trong bảng — dùng trước khi vẽ lại."""
        for item in self.tree.get_children():
            self.tree.delete(item)

    def them_dong(self, sp_dict):
        """Thêm 1 dòng vào bảng từ dict sản phẩm."""
        rieng = self._str_truong_rieng(sp_dict)
        gia_cb  = f"{sp_dict.get('gia_co_ban', 0):,.0f}"
        gia_ban = f"{sp_dict.get('gia_ban', 0):,.0f}"
        self.tree.insert('', 'end', values=(
            sp_dict.get('ma_so', ''),
            sp_dict.get('ten_san_pham', ''),
            sp_dict.get('loai_hang', ''),
            gia_cb,
            sp_dict.get('ton_kho', 0),
            gia_ban,
            rieng
        ))

    def lay_ma_dang_chon(self):
        """Trả về mã số của dòng đang được chọn. None nếu chưa chọn."""
        selected = self.tree.selection()
        if not selected:
            return None
        values = self.tree.item(selected[0], 'values')
        return values[0]   # values[0] là cột "Mã số"

    def _str_truong_rieng(self, sp):
        """Tạo chuỗi hiển thị thông tin riêng theo loại hàng."""
        loai = sp.get('loai_hang', '')
        if loai == 'Sách':
            return f"Tác giả: {sp.get('tac_gia', '')} | NXB: {sp.get('nha_xuat_ban', '')}"
        elif loai == 'Tạp chí':
            return f"Số phát hành: {sp.get('so_phat_hanh', '')}"
        elif loai == 'Báo giấy':
            return f"Ngày XB: {sp.get('ngay_xuat_ban', '')}"
        elif loai == 'Luận văn':
            return f"Tác giả: {sp.get('tac_gia', '')} | Trường: {sp.get('truong_dai_hoc', '')}"
        elif loai == 'Bản thảo':
            return f"Tác giả: {sp.get('tac_gia', '')} | TT: {sp.get('tinh_trang', '')}"
        return ''

4. File _dk: DieuKhienDocDanhSach

from .doc_danh_sach_tt import lay_danh_sach
from loi_thuat_toan.danh_sach_lien_ket import DoublyLinkedList


class DieuKhienDocDanhSach:
    """Lớp điều khiển — gắn kết UI và thuật toán đọc danh sách."""

    def __init__(self, ui_bang, danh_sach: DoublyLinkedList):
        self.ui = ui_bang
        self.danh_sach = danh_sach

    def nap_lai(self):
        """Đọc toàn bộ danh sách từ DSLK → vẽ lại bảng."""
        ds = lay_danh_sach(self.danh_sach)
        self.ui.xoa_tat_ca_dong()
        for sp in ds:
            self.ui.them_dong(sp)
        return len(ds)   # Trả về số lượng để caller hiển thị thanh trạng thái

    def hien_thi_danh_sach(self, ds_dict):
        """Hiển thị 1 list dict bất kỳ lên bảng (không tải lại từ DSLK)."""
        self.ui.xoa_tat_ca_dong()
        for sp in ds_dict:
            self.ui.them_dong(sp)
        return len(ds_dict)

    def lay_ma_dang_chon(self):
        """Trả về mã số dòng đang chọn — dùng cho Sửa/Xóa/Thêm vào giỏ."""
        return self.ui.lay_ma_dang_chon()

2 hàm khác nhau:

  • nap_lai(): đọc lại từ DSLK toàn cục → vẽ lại bảng. Dùng sau khi thêm/sửa/xóa/undo/redo.
  • hien_thi_danh_sach(ds_dict): hiển thị list dict tùy ý (không tải từ DSLK). Dùng cho kết quả tìm kiếm/sắp xếp.

5. Cú pháp Treeview

5.1. Tạo Treeview

self.tree = ttk.Treeview(parent, columns=COT, show='headings', height=15)

Các tham số quan trọng:

  • parent: widget cha chứa Treeview.
  • columns: tuple tên các cột (VD: ('ma', 'ten', 'loai')).
  • show='headings': chỉ hiển thị phần header — ẩn cột #0 (cột cây mặc định của Treeview). Nếu muốn Treeview hoạt động như cây thư mục thì dùng show='tree headings'.
  • height=15: số dòng hiển thị (15 dòng). Nếu nhiều hơn, sẽ có scrollbar cuộn.

5.2. Định nghĩa header + cột

# Đặt tên hiển thị cho header
self.tree.heading('ma', text='Mã số')
self.tree.heading('ten', text='Tên sản phẩm')

# Cấu hình cột: độ rộng + căn lề
self.tree.column('ma', width=70, anchor='center')
self.tree.column('ten', width=200)
self.tree.column('gia_cb', width=100, anchor='e')

anchor (căn lề trong cell):

  • 'w': trái (West)
  • 'e': phải (East) — dùng cho số tiền
  • 'center': giữa
  • 'n', 's': trên, dưới

5.3. Thêm dòng

self.tree.insert('', 'end', values=('MH001', 'Truyện Kiều', 'Sách', 50000, 20, 55000, 'Tác giả: Nguyễn Du'))
  • Tham số 1 (''): ID của Node cha. '' = root (dòng cấp cao nhất).
  • Tham số 2 ('end'): vị trí — 'end' = cuối danh sách. Có thể dùng '0', '1' để chèn vào vị trí cụ thể.
  • values=(...): tuple chứa giá trị các cột theo thứ tự đã khai báo trong columns.

5.4. Lấy danh sách dòng

# Lấy tất cả ID dòng trong Treeview
for item in self.tree.get_children():
    self.tree.delete(item)   # Xóa từng dòng

get_children() trả về tuple các ID dòng. Để xóa toàn bộ, lặp qua từng ID và gọi delete(item).

5.5. Lấy dòng đang chọn

selected = self.tree.selection()      # Tuple ID dòng đang chọn
if not selected:
    return None
values = self.tree.item(selected[0], 'values')   # Lấy tuple values
ma_so = values[0]                                # values[0] = cột đầu tiên

Lưu ý: selection() trả về tuple (vì Treeview hỗ trợ chọn nhiều dòng). Muốn lấy 1 dòng đầu tiên, dùng selected[0].

5.6. Scrollbar cho Treeview

# Tạo scrollbar dọc
scrollbar = ttk.Scrollbar(parent, orient='vertical', command=self.tree.yview)
# Kết nối: khi cuộn scrollbar → cuộn tree
self.tree.configure(yscrollcommand=scrollbar.set)

# Đặt widget: tree trái, scrollbar phải
self.tree.pack(side='left', fill='both', expand=True)
scrollbar.pack(side='right', fill='y')

2 dòng kết nối 2 chiều:

  • command=self.tree.yview: khi user kéo scrollbar → tree cuộn theo.
  • yscrollcommand=scrollbar.set: khi tree cuộn (bằng phím, mouse wheel) → scrollbar cập nhật vị trí.

6. Format tiền: f-string {x:,.0f}

Trong them_dong, có dòng:

gia_cb = f"{sp_dict.get('gia_co_ban', 0):,.0f}"
# 50000 → '50,000'
# 1200000 → '1,200,000'

Giải thích cú pháp format

f"{x:,.0f}" là f-string với format spec:

  • ,: thêm dấu phẩy hàng nghìn (1000 → 1,000).
  • .0f: làm tròn 0 chữ số thập phân (f = float).

Các ví dụ khác:

x = 1234567.891

f"{x:,.0f}"     # '1,234,568'      (0 số thập phân)
f"{x:,.2f}"     # '1,234,567.89'   (2 số thập phân)
f"{x:,.2e}"     # '1.23e+06'       (ký hiệu khoa học)
f"{x:>15,.0f}"  # '      1,234,568' (căn phải, rộng 15 ký tự)
f"{x:<15,.0f}"  # '1,234,568      ' (căn trái)
f"{x:^15,.0f}"  # '   1,234,568   ' (căn giữa)
💡 Hàm format() cũng tương tự

Nếu không dùng f-string, có thể dùng hàm format():

gia_cb = format(50000, ',.0f')   # '50,000'
# Tương đương: f"{50000:,.0f}"
